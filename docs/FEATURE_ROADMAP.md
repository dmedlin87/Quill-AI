# Feature Roadmap
Date: 2024-05-24

We proposed 5 must-have features. We are picking the best one to ship first, but tracking the others here for future implementation.

## 1. Export Studio (STATUS: COMPLETE)
**Goal:** Allow users to export their work in standard formats.
- [x] PDF Export (with formatting options)
- [x] DOCX Export (standard manuscript format)
- [x] Markdown/Text Export
- [x] Export Modal UI with options (include title page, chapters, etc.)

## 2. Character Voice Dashboard
**Goal:** A visual dashboard to track character consistency.
- [ ] Voice metrics per character
- [ ] Dialogue distribution charts
- [ ] Consistency alerts list

## 3. "Dream Mode" (Deep Sleep Analysis)
**Goal:** Background processing when the user is away.
- [ ] Detect idle state > 5 minutes
- [ ] Run deep consistency checks on all chapters
- [ ] Generate a "Morning Report" of insights

## 4. Plot Hole Detector
**Goal:** Dedicated analysis for narrative logic.
- [ ] Timeline inconsistencies
- [ ] Dead-end subplots
- [ ] Character motivation gaps

## 5. Zen Mode + Focus Analytics
**Goal:** Distraction-free writing with productivity stats.
- [ ] Fullscreen editor with no UI
- [ ] Session word count goals
- [ ] Focus timer
